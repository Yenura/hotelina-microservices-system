# Restaurant Service

Restaurant Service is part of the Hotelina Microservices System.
It provides food menu management and restaurant order management.

## Requirements
- Node.js
- MongoDB

## Install
```bash
npm install
```

## Run
```bash
npm start
```

## Development
```bash
npm run dev
```

## Environment
Copy `.env.example` to `.env` and update values.

## API
- Service port: `5004`
- Swagger docs: `http://localhost:5004/api-docs`

## Notes
This service is ready to integrate with an API Gateway.
